# RFT-Auth-App

Login-register app with react, firebase and tailwind

## Start
```bash
create-react-app app --template typescript
```
## Tailwind
```bash
npm install tailwind postcss autoprefixer
npx tailwindcss init -p
```
## tailwind.config:
```json
content: ["./src/**/*.{js,jsx,ts,tsx}"]
```
### index.css:
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
